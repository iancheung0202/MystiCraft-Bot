Workflow:

1. Download entire codebase from [backup](https://panel.mysticraft.xyz/server/25fe5a4d/backups)
2. Extract `tar.gz` file and rename folder to `MystiCraft-Bot`
3. Run the following command in the `MystiCraft-Bot` folder
```
git init
git remote add origin https://github.com/iancheung0202/MystiCraft-Bot.git
git fetch
git reset --mixed origin/main
```
4. Make edits, copy to server, commit and push to Github
